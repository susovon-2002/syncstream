# **App Name**: SyncStream

## Core Features:

- Room Creation/Joining: Users can create a room and share the room ID with others, or join an existing room using its ID.
- Video/Audio Playback Sync: Synchronize video and audio playback across all connected clients in a room, using WebSocket-based real-time communication.
- Chat Integration: Real-time chat functionality within each room for users to communicate while watching/listening together.
- URL Preview: Generate a title, description and image representing the URL a user submits for shared viewing, with the help of a tool.
- Video/Audio Upload and Hosting: Allow users to upload their own video/audio files to be hosted and streamed within the platform.

## Style Guidelines:

- Primary color: Electric purple (#BE29EC) for a modern, vibrant feel.
- Background color: Very dark purple (#16041D), almost black, to provide a high-contrast, immersive experience.
- Accent color: Neon green (#39FF14) for interactive elements and highlights.
- Body and headline font: 'Space Grotesk', a sans-serif with a techy, scientific feel that works well for headlines and body text. 
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean, minimalist icons for playback controls, room management, and chat functions.
- Subtle animations and transitions for interactive elements, such as play/pause button feedback and chat message animations.
